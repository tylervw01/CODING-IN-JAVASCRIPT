JavaScript Essentials 1(JSE):

MODULE 4

Control Flow - Conditional Execution and Loops

After completing Module 4, the student will: be able to force conditional execution of a group of statements (make decisions and branch the flow) using if-else and switch commands; be able to force a group of statements to repeat in a loop using the for, while, and do-while commands, using both dependent and independent conditions on the number of iterations; understand and be able to use loop-specific break and continue instructions; be able to use the for-in statement to iterate over the properties of an object; be able to use the for-of statement to walk through the elements of an array.
